Design Research regarding the possibilities of using AI to generate vehicle acoustics, with sonification.

<img width="1000" alt="osc_receiver_chain" src="https://github.com/user-attachments/assets/8aace748-2ec4-4250-a70b-56d958912872" />

These Values from the OSC serve as input for the .pd patch which loops, and alters the sound of the input .wav file.

<img width="1000" alt="synthesis_engine_chain" src="https://github.com/user-attachments/assets/211e097d-198b-4f9d-9bf3-5d0cdc4d667e" />

From the /input folder, template .wav files can be found which can serve directly as input for the .pd patch. It is recommended to create your own 5 - 10 sec samples for a personalized experience.
